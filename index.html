What is an Arithmetic Sequence?

An arithmetic sequence is a sequence of numbers where the difference between the consecutive terms is constant. For example, the sequence 1, 4, 7, 10, 13, 16... is an arithmetic sequence with a common difference of 3.

How to find the nth term of an Arithmetic Sequence?

Given an arithmetic sequence with the first term (a1) and the common difference (d) , the nth term (an) of the sequence is given by an=a1+(n−1)d .

Sum of an Arithmetic Sequence

The sum of the arithmetic sequence formula is used to calculate the total of all the terms present in an arithmetic sequence. Given the series {1,2,3,4,5,6} we notice that by adding the first to the last term, the second to the second to last term and so on, we get 7 as a constant sum for each pair of terms. The formula gets this sum, represented by (a1 + an) and multiplies it by the number of pairs in the sequence, represented by (n/2). In this case the number of pairs is 3, so 3*7=21, which is the sum of the series. 

By substituting the last term (an) for the formula in finding the nth term, we are able to derive a formula without any given last term.

Relation to Linear Functions 

By graphing the values of any arithmetic sequence, we can see that it has some similarities to linear equations. The slope is the common difference, the x-value could be represented by the index of a term or 'n', and the y-intercept is given by finding the value when n is 0 (a0 = a1-d)
